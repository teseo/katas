# katas
PHP katas with BDD/TDD Gulp and PHPSpec including:
- Roman numbers translator
- Bowling game
- Prime numbers calculator

#Usage

To run all the test
```
gulp
```
To run the test suite for each individual kata:
- Bowling Kata:
```
gulp test-bowling
```
- Prime Numbers Kata:
```
gulp test-prime
```
- Roman Numbers Kata:
```
gulp test-roman
```
